map to get an array of product prices.
reduce to get the total cost of all the products.
JavaScript
Copy
Input: [ {name: 'Apple', price: 0.5, quantity: 15}, {name: 'Banana', price: 0.25, quantity: 5}, {name: 'Cherry', price: 1, quantity: 12} ] Output: 10.5 (because 0.5*15 + 1*12 = 10.5)
