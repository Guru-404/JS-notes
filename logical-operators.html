Logical Operators

to combine expressions
what are expressions?
a < b
now there’s a second expression
b < c
now to combine the values of these two separate expressions, logical operators are used.
now both these expressions either return true or false value.
So whenever from two expressions, one returns some value & another returns some value from true or false, and we want to combine them and check whether the result should come true or false
for e.g.
we’ve enrolled in the course & we’ve completed this course.
so to get the completion certificate, we’ve to both enroll & complete this course.
We’ve primarily 3 types of logical operators.
&& → Logical AND
|| → Logical OR
! → Logical NOT
